# Realtime OpenCode Workspace Monitor

Real-time activity dashboard for OpenCode agents, sessions, and system resources.

## Features

- **Office Floor** — see active worker processes on numbered desks
- **Sessions** — real-time task cards with state, cost, and token usage
- **Rest Room** — idle staff relaxing with activities
- **Activity Log** — track transient commands and session events
- **System Stats** — CPU, memory, disk, and total cost tracking

## Usage

```bash
bash start.sh
# Opens http://localhost:5500
```

## Data

The dashboard auto-refreshes every 2 seconds. No database required.
